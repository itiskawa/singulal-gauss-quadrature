Detailed Description

template<typename T>
class GQLog::GaussLogRule< T >

A class that contains the nodes, weights and number of points of a Gaussian quadrature rule with weight function w(x)=ln(1/x) over ]-1,1[ It inherits from the GaussRule<T> and is also a class template qith parameter T.

The modified Chebyshev Algorithm. It computes the recurrence relation coefficients of the orthogonal polynomial w.r. to w(x) = ln(1/x) The mathematics of the computation are explained in Chapter 2 of W. Gautschi's 'Orthogonal Polynomials: Computation and Approximation'.

Parameters
n: the desired amount of recurrence relation coefficients
mom: the 2n first modified moments
abj: the 2n first shifted recurrence relation coefficients
Returns
The recurrence relation coefficients of the orthogonal polynomial w.r. to w(x) = ln(1/x)

Applies quadrature rule to a function This evaluation is a bit tricker, as the desired weight function is actually ln(x+1) over ]-1,1[, not ln(1/x) over ]0,1[ Because of this, half of the quadrature is solved via a GaussLegendreRule, and the other half by this.

Parameters

f: function template
Returns
numerical value equal to the approximation of I[f,ln(x+1)]
