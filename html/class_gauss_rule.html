Detailed Description

template<class T>
class GaussRule< T >

An abstract class that defines common functions to different quadrature solvers It has been chosen to not contain fields nor evaluation operators, due to subtleties in subclasses (notably GaussLogRule<T>)

Computes the recurrence relation coefficients of the monic Jacobi Polynomials The methods mimics r_jacobi.m of W. Gautschi in his MATLAB Suite for Orthognonal Polynomials available at: https://www.cs.purdue.edu/archives/2002/wxg/codes/OPQ.html.

Parameters
n: desired number of coefficients, should match the number of quadrature nodes / weights for a Gauss-Jacobi quadrature rule
a: the a parameter of the Jacobi weight function
b: the b parameter of the Jacobi weight function
Returns
a nx2 array with the recurrence relation coefficients in the columns

rearranges an array of recurrence relation coefficients into a self-adjoint matrix for Golub-Welsch solve The alpha coefficients are placed in the diagonal, and the n-1 first beta coefficients' square roots are placed tridiagonally

example with n=4: 

 | a_0 c_1 0    0   0  |
 | c_1 a_1 b_2  0   0  |

J = | 0 c_2 a_3 c_2 0 | where c_i = sqrt(b_i) | 0 0 c_3 a_4 c_3 |

Parameters
coeffsnx2 array of recurrence relation coefficients, n: number of desired coefficients
n: desired matrix size, should match the number of coefficients
Returns
nxn self-adjoint tridiagonal matrix
