Detailed Description

template<class T>
class GQJacobi::GaussJacobiRule< T >

A class that contains the nodes, weights and number of points of a Gauss-Jacobi quadrature rule It inherits from the GaussRule<T> and is also a class template qith parameter T.

template<class T >
Eigen::Matrix< T, Eigen::Dynamic, Eigen::Dynamic > GQJacobi::GaussJacobiRule< T >::nw ( )
inlineprotected

Computes the nodes and weights of the Gauss-Jacobi quadrature rule The nodes and weights are computed via the Golub-Welsch algorithm using the recurrence relation coefficients of the monic Jaocbi polynomials. The eigenvalue problem is solved using a self-adjoint matrix solver, which significantly speeds up the computation.

Returns
a (nx2) array with then nodes & weights in the columns
